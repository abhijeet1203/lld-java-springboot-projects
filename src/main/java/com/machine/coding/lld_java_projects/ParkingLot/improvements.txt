| Category       | Pattern                 | Role in Parking Lot            | Usecases                     |

|                | Abstract Factory        | Not valid                      |                              |
|                | Builder                 | Valid                          | Booking objects, result & failed results objects
| Structural     | Decorator               | Not valid                      |                              |
|                | Adapter                 | Not valid                      |
|                | Facade                  | Valid                          | ✅ Already used at ParkingLotApplication and ParkingController

|                | State                   | Not valid for now              | Needed if want to switch between states, avoids if-else or switch
